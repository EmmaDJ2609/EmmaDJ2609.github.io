Uitleg binairstelsel

Wij gebruiken zelf het decimale talstelsel of het tientallig talstelsel. Dit talstelsel maakt gebruik van de tien getallen 0 t/m 9. Als je gebruik wilt maken van bijvoorbeeld het getal 10 dan komt er een nieuwe rij getallen links van de eerste getallen die er al stonden. Als je naar nog een getal wilt zoals het getal 100 dan komt er nog een getal links voor te staan. Op het moment dat dat gebeurd gaan de getallen die er eerst stonden weer terug naar nul. Het binairstelsel is een getallenstelsel dat uit twee getallen bestaat. De één en de nul. Deze getallen worden ook wel een bit genoemd. Een reeks van acht van deze twee getallen heet een byte. Een byte lees van links naar rechts. Computers maken gebruik van het binairstelsel.

Bij dit binairstelsel maak je gebruik van machten van 2. Je begint rechts met 2⁰. Daar komt 1 uit. Daarna 2¹. Daar komt 2 uit. Zo ontstaat er rijtje van acht getallen met machten van twee. Het hele rijtje staat in de afbeelding. Het gaat in het plaatje van 2⁰ t/m 2⁷. Daar komt namelijk 128 uit. Het gaat maar door tot 128, omdat je computer gebruikt van 8 byts.

Voorbeelden

Stel je hebt de code 110 die lees je van rechts naar links. De eerste in het rijtje is 2⁰ die staat op nul, dus die doe je keer 0: 2⁰ x 0. De tweede in het rijtje is 2¹. Daar staat een 1 bij, dus dan is het:2¹ x 1. De derde is 2² deze staat ook op 1, dus dan wordt het: 2² x 1. Dit tel je bij elkaar op en dan krijg je je eindantwoord: 2⁰ x 0 + 2¹ x 1 + 2² x 1 = 6

In het tabel hiernaast staat ook een code van 8 getallen. je ziet dat er bij de 2, 4, 16 en 64 1 staat. In het tabel hebben ze de machten al uitgewerkt, dus kan je het gewoon optellen: 2+4+16+64 =86 Met machten zou het worden: 2⁰ x 0 + 2¹ x 1 + 2² x 1 + 2³ x 0 + 2⁴ x 1 + 2⁵ x 0 + 2⁶ x 1 + 2⁷ x 0 = 86
